### 公安大数据系统
* 高性能并行计算集群

主要负责搭建一个集群管理和资源调度的系统。当前主要并行计算平台都有自己的一套任务调度算法，而工程越来越细化和复杂化，只使用一个并行计算平台已经无法满足要求。为此，根据每个模块的特性和要求，选择适合的并行计算平台。由此带来的各平台的不兼容性，整个工程的任务和资源无法统一调度。本系统在实现整个系统的高性能的同时，对任务调度进行抽象，封装出统一接口，实现对多个并行计算平台的任务和资源调度。
